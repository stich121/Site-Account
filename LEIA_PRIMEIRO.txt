SE ALGO NAO FUNCIONAR
================================================================================

Problema 1: Ao clicar em "PDF", nada acontece
  → Verifique se MySQL está rodando
  → Verifique credenciais em tools/gerar_folha_ponto_pdf.py
  → Verifique logs do servidor web

Problema 2: Gera HTML em vez de PDF
  → Normal! WeasyPrint não pode converter em PDF no Windows
  → Abra o HTML no navegador: Ctrl+P > Salvar como PDF
  → Resultado é identico

Problema 3: Python não encontrado
  → Verifique: python --version
  → Se não funcionar, reinstale Python de https://www.python.org
  → IMPORTANTE: Marque "Add Python to PATH" durante instalação

Para mais troubleshooting:
  → Abra: tools/README_GERAR_FOLHA_PONTO.md (seção Troubleshooting)
